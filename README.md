# distraction-free-youtube
A simple HTML based project designed to provide a distraction-free video viewing experience:
About the Project
This project offers a clean and minimal alternative to traditional video platforms by focusing only on selected content categories. Instead of endless recommendations and distracting elements, users can directly access videos from predefined categories.
Features
*Clean and simple interface
*Distraction-free browsing experience
*Embedded YouTube videos for direct viewing
*Organized content into three categories:
 1.Music
 2.Gaming
 3.Cartoon
*Easy navigation between categories
Technologies used
*HTML5
Project Objective
The goal of this project is to demonstrate how a focused video platform can help users access content without unnecessary distractions.
Future Improvements
*Add CSS styling for a modern user interface
*Make the website responsive for mobile devices
*Add search and filtering functionality
*Introduce dark mode
*Add personalized content preferences
Author
Created as a frontend web development project to practice HTML structure, navigation, and embedded media integration.
